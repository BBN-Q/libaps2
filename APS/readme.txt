The directory contains the standalone APS software.
It includes:

bitfiles - latest bitfiles for DACII or APS hardware
examples - example individual channel waveforms and 4 channel sequence files
python - python interface
libaps-cpp - C++ library used by both the python and matlab interfaces

The MATLAB interface is stored elsewhere. It can be found at QLABROOT/common/src/+deviceDrivers/@APS.

Python Requirements
-------------------

The python interface has been tested on Windows 7 using Python 2.7. It requires

pyside: http://www.pyside.org/
numpy:  http://numpy.scipy.org/
h5py:   https://code.google.com/p/h5py/
